ZKP-Enhanced Authentication Framework

A secure authentication system demonstrating Zero-Knowledge Proof inspired verification, context-bound password reset, replay attack prevention, and deterministic session revocation.

🔐

ZKP-Style Proof

Password reset requires cryptographic proof using HMAC-SHA256. The password is never transmitted.

🔄

Replay Prevention

Each reset increments the lifecycle counter, making all previous proofs invalid.

📱

Multi-Factor Auth

Password reset requires both OTP verification and cryptographic proof.

🚫

Session Revocation

After password reset, all existing sessions are immediately invalidated.

📊

Lifecycle Tracking

Every authentication state change is tracked and visible.

🛡️

Context-Bound Reset

Password reset is bound to the user's current lifecycle state.

🚀 Getting Started

  1. Register — Create account (reset_counter=0, session_version=1)
  2. Login — Get JWT token with session_version
  3. Dashboard — View protected session and state
  4. Password Reset — OTP → Verify Proof → Reset
  5. Security Logs — View all security events
